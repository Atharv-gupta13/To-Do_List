# 📝 To-Do List App

A simple and modern To-Do List application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Tasks saved using Local Storage
- Responsive design
- Clean and user-friendly UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
